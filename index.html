Our Work

Crypto Conway

Crypto Conway is an NFT project where you can create HTML based NFTs by creating seeds for Conway's Game Of Life using custom ERC721 smart contracts (NFT Collection):

- Written in Solidity with MetaMask integration
- Truffle as the framework
- On an Ethereum commit chain called Polygon
- Storing the whole 'seed' onchain
- Generated live from a Node.js server

Bayer Innovation

Bayer Innovation team was seeking a web3 partner to audit and document an entire value chain so we could identify opportunities where  blockchain-based technology could be utilized to the benefit of the seed producer, the reseller, the grower, the processor and/or the financial institutions supporting the transactions.

Our team mapped the entire value chain by user segment, phase and detailed use case descriptions to support the identification, segregation and traceability of the product’s unque, value-added characerstics across the entire value chain.

DoodlesData.com 

DoodlesData is a web app built using  React/Node JS to help visualize sales, floor price, etc. about a preexisting NFT collection called Doodles.

Integrating with  OpenSea's API to pull listing information for each NFT and storing for sorting/filtering.

Also uses Snapshot's API (a governance protocol) to pull in proposals to encourage more engagement.
